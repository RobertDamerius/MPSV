Select Current Point On Trajectory

Library

MPSV / Select Current Point On Trajectory

Description

Select the current point on a given trajectory for a given timepoint. An arbitrary time source can be selected by the user. However, all times must be given in seconds and must be monotonically increasing.

Input Ports

timeNow

The current timepoint to be used to query the point on the given trajectory.

trajectoryStartTimepoint

The timepoint that indicates the first data point of the trajectory.

trajectorySampletime

The temporal distance between two points of the trajectory.

trajectoryNumPoints

The number of points to be considered from the trajectoryData matrix.

trajectoryData

A 12-by-500 matrix containing 500 points of the trajectory, where each point represents the state-input-vector [x; y; psi; u; v; r; X; Y; N; Xc; Yc; Nc].

Output Ports

success

True if timeNow lies within the temporal range of the trajectory and a point can be selected, false otherwise.

currentPoint

A bus containing the linear interpolated state values and the zero-order hold input values. The elements of the bus are given in the table below.
ElementData TypeDescription
stateInterpolated.positionNorthdoubleThe north position (x) in meters.
stateInterpolated.positionEastdoubleThe east position (y) in meters.
stateInterpolated.orientationYawdoubleThe heading angle in radians.
stateInterpolated.velocityUdoubleThe body-fixed surge velocity in meters per second.
stateInterpolated.velocityVdoubleThe body-fixed sway velocity in meters per second.
stateInterpolated.velocityRdoubleThe angular velocity in radians per second.
stateInterpolated.tauXdoubleThe longitudinal force in x-direction of the body frame in Newton.
stateInterpolated.tauYdoubleThe lateral force in y-direction of the body frame in Newton.
stateInterpolated.tauNdoubleThe torque around the z-axis of the body frame in N*m.
controlInputZeroOrderHold.commandedTauXdoubleThe commanded longitudinal force in x-direction of the body frame in Newton.
controlInputZeroOrderHold.commandedTauYdoubleThe commanded lateral force in y-direction of the body frame in Newton.
controlInputZeroOrderHold.commandedTauNdoubleThe commanded torque around the z-axis of the body frame in N*m.

remainingTrajectoryDuration

Time from timeNow to the end of the trajectory.
