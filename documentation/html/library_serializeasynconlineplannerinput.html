Serialize AsyncOnlinePlanner Input Bus

Library

MPSV / Serialize AsyncOnlinePlanner Input Bus

Description

Serialize an AsyncOnlinePlanner input bus to a binary representation.

Input Ports

asyncOnlinePlannerInput

The input bus for the asynchronous online planner, which is constructed via the Default AsyncOnlinePlanner Input Bus block.

Output Ports

bytes

Array of uint8 values representing the bytes of the serialized bus.

length

Number of bytes representing the serialized data.
